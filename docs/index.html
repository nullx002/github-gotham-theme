Gotham Theme

Very Dark theme for Github, with color palette directly adopted from Gotham Theme for VIM and EMACS. This is a minimalistic theme with flexible width and no borders for all most all elements. If you prefer narrow (regular) width with borders, remove the code at the bottom of .css file as commented.
